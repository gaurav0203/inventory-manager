# Inventory Manager Application

### Highlights

-   Login, Register, Delete Users
-   Admin, Manager, Staff Roles
-   Add, Delete, Update Product
-   Update Stock
-   Dashboard
-   Transaction History

### Technologies/Libraries Used

-   Python
-   Flask, Flask-login
-   SQLite
-   Bootstrap
